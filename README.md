# Unknown Seas
A simple text-based Navel combat game, Players will fight through two short levels using an input box to give commands

## Features
-5 diffrent Aircraft carriers to chose from
-Two diffrent levels
-Input reads commands and exicutes them
-Log provides feadback
-Game resets at game over or level completion

## How to play
-Commands:
Move: move between the levels, entering the name, or go and then the name works
Attack: using target number of fighters number of divebombers number of torpedobombers to damage enemies
Repair: heals the player and recovers some of their aircraft, after use enemies must attack before it can be used again
Scout/Support: non-combat objectives,  having a chance to succeed or fail every attempt
End: ends the level if all objectives are completed
